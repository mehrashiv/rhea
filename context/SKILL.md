Key facts:
- One brand hue: lilac #7E6CD4 over neutral ink/paper. No second accent.
- Type: Cormorant Garamond (serif, display/wordmark) + Hanken Grotesk (sans, body/UI).
- Soft pill buttons, 16–20px card radii, low lilac-tinted shadows, gentle 200ms ease-out motion.
- Voice: warm, plain, first-person maker. Sentence case; lowercase wordmark `lila`. No hype, minimal emoji.
- Surfaces: link `styles.css` and use the CSS custom properties (never hard-code hex/fonts).
